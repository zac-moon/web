Domain Structure

example.com/
----bot
----owndet
----setup
----index.html
----more/
--------index.html

About Each Bit:
•bot - The bot file is used by search engines to find key words related to the site in question. It contains single words in seperated by blank spaces.
•owndet - The owndet file is used to claim a site to a Company or Person. It's normally just Company Inc or Firstname Lastname or Company Inc with Firstname Lastname.
•setup - The setup file describes how the site was originally created. It is always either a Site Creation program name or just "self".
•index.html - The index.html file is the html content that is displayed when you run the Straight Domain of the url request.
•more/index.html -This is a subdomain and the subdomain's index.html file. The folder name is the name of the subdomain, and the index.html file inside of it is the content to be shown when it's url is called.

HTML and Languages

All sites that run on linker and web are powered by HTML5, and integration with CSS and JS. However only some html elements work. List of tested working HTML elements:
•h1,h2,h3,h4,h5,h6
•p
•button
•title
•br
•body
•head
•!doctype
•style
•script

However, more html elements may work but cannot be garunteed. All javascript functions work, but not window. and other frontend funtions. alert() works too.

Python GUI Integration

If you want to enhance your development journey, you can set up Python GUI Programs to run when domains/subdomains/subsubdomains etc. are opened. The .py GUI file must be checked by web.com before being added as a Linker method.

Tutorial

To set up your website, begin by navigating by the url bar to eng.start.web.com. When you load this site, you should have a popup window appear. Fill in all the details required and press "Start". From there the program will create all the files required for your basic site.
